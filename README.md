# Music Player

## Language
- Python

## Features
- Plays music in a simple yet aesthtic format
- Able to skip a song, go back a song, pause the music or pick a place in the song to play from
- Displays a time bar showing the progress of the song
- Loops back to beginning of song list if skipped through all
